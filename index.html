Grid Template

Då man skapar en layout med grid används bl.a. CSS-egenskaper grid-template-columns, grid-template-rows och grid-template-areas. Beskriv vad dessa används till.

För att anpassa mätningar på kolumner med mall egenskaper skapat i CSS kan man använda sig av olika anpassningar. Två specifikationer som kan användas i detta syfte är grid-template-rows (som specificerar höjderna på varje kolumn) och grid-template-columns (som specificerar bredderna på varje kolumn). På bilden visas hur första kolumnen tilldelas siffran 1 och det sista spåret tilldelas siffran -1 och räknas fortsättningsvis baklänges, det vill säga -2, -3 etc.

Man kan också namnge områden genom att använda sig av grid-template-areas. Varje cell i rutnätet tilldelas då ett namn som listas inom citationstecken rad för rad. Närliggande celler med samma namn bildar ett område i rutnätet. Ett rutnät som ska bestå av tre kolumner måste ha tre olika namn för att bilda tre separata kolumner. Man kan även lämna en cell namnlös genom att skriva en eller flera punkter (.) för att fylla utrymmet och således få med varje cell. Främst används grid-template-area för att man lätt skall kunna placera ut objekt i layouten vid ett senare skede.

Media Queries

Vad är en media query och vad används den till?

De finns ett flertal steg som används för att skapa en responsiv design en av dessa steg är media query som används för att skapa förändringar i form av stilregler som behövs för att anpassa layouten för olika storlek på webbläsarens fönster. När man arbetar med responsiv design kan det vara bra att visualisera layouten med hjälp av skisser. Media queries används för att sätta ut gränser i layouten. På så sätt anpassas sidans innehåll efter tillskrivna nya regler när en gräns för det tidigare innehållet passeras. Ett exempel kan vara att man anpassar en webbsidas bredd till en smalare layout med en bredd på 600 pixlar. Då kan man anpassa webbplatsens innehåll med media queries som skapar nya regler för den alternativa webblayout som skapas när man passerar en gräns på en bredd på 600 pixlar. När man gör om koden på detta vis måste man utgå från den de regler som man använt i den ursprungliga layouten. Man markerar således ut de regler som man vill anpassa till den nya layouten. Ett exempel på en media query som skapats för att göra layouten responsiv kan ses i bild 1. Anpassningar kan ske genom elegant nedbrytning eller gradvis utökning.
