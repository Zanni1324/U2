Global och lokal navigering

Beskriv vad global respektive lokal navigering är?

Navigering används för att besvara frågorna var är jag samt hur kommer jag dit jag vill? I cyberspace är navigationsfunktionen ett verktyg som hjälper användaren att, med minst antal klick, navigera sig fram och tillbaka på en webbplats. Koncentrerad vertikal och horisontell navigation har använts för att skapa flexibla strukturer för huvudmenyer och dess underkategorier i form av så kallade “dropdown”-menyer som utvecklats för att skapa en känsla av minimalism, smart skärmanvändning och enkel navigering.

De flesta moderna webbsidor använder sig av global navigering som hjälper användaren att navigera sig mellan webbplatsens avdelningar. Den globala navigationsmenyn bör alltid finnas tillgänglig oavsett var användaren befinner sig på webbsidan. Lokal navigering används för att användaren ska kunna söka i informationen inom en avdelning eller på en specifik sida.

Supplementary och Courtesy navigering

Beskriv vad supplementary respektive courtesy navigering är.

Ytterligare två sätt att navigera på en webbplats är supplementary navigation och courtesy navigation. Supplementary navigation baseras på att man skapar genvägar till relaterat innehåll.

Courtesy navigation tar upp länkar till information som kan anses sekundär, men skall fortfarande existera och vara enkelt att få tag på när användaren är i behov av den.

Exempel på olika typer av navigering

För att visa upp ett exempel på en webbsida med en global navigering kan man titta på Plantagen där man kan webbplatsens olika avdelningar listas i headern som vecklas ut till en mega meny då användaren håller pekaren över de olika alternativen.

För att titta på ett exempel som visar på lokal navigering kan man söka upp Lyko. Precis som på Plantagen använder webbplatsen sig av en global navigeringsmeny. I den globala navigeringsmenyn kan användaren klicka sig in på sidans olika avdelningar som “ansikte” där användaren sedan på nytt kan söka sig genom innehållet via den lokala navigeringen som visar på varje subkategori inom varje respektive avdelning på webbplatsen.

För att visa upp ett exempel på supplementary navigation har jag använt mig av webbplatsen Reuge Genom att klicka in på fliken “shop” och klicka sig vidare via fliken “boutique and retailers” får man upp en lista på alla återförsäljare samt en karta som utifrån användaren position visar vart man kan besöka en fysisk butik. Således ett exempel på hur man kan använda supplementary navigation för att skapa genvägar till relaterat innehåll.

För att visa på ett exempel på courtesy navigering har jag valt att stanna på https://www.reuge.com/ och skrollat längst ner till footern där man kan se kontaktuppgifter och adressuppgifter som gör det möjligt för användaren att komma i direkt kontakt med tillverkaren.
